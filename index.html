RULES

1. Press the START button.

2. You will receive a signal.

3. Repeat the signal by pressing the same color.

4. It will duplicate the first signals and add one.

5. Repeat these two signals by pressing the same color button, in order.

6. Continue playing as long as you can repeat each sequence of signals correctly.

7. If you fail to repeat a sequence exactly, you will hear a "WRONG" sound effect plus RED screen.

8. Press RESTART to start the game again.
